# Questionnaire

## 서비스 소개

수업, 발표, 컨퍼런스, 데모데이에 발표자에게 청자(익명가능)들이 실시간으로 질문을 하고 발표자는 해당 질문을 발표자 화면에서 정렬시켜 확인 응답할 수 있게 도와주는 서비스

## 서비스 기능

- 이벤트별 질문방 개설가능
- 청자들이 생성한 질문에 좋아요 버튼을 통해 질문의 선호도 확인 가능
- 청자가 생성한 질문을 발표자는 특정 우선순위(생성순, 좋아요순)로 구분하여 볼 수 있음
